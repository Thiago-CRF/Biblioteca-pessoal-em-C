Progama simples em C que gerencia livros cadastrados para funcionar de bilbioteca de uso pessoal para gerenciar livros adquiridos e o progresso de leitura.

Salva os dados em um .txt, que no caminho para o fopen coloquei para buscar por um pasta "data" (data/livros.txt), se essa pasta não existir no local do executavel
o programa não funciona.
